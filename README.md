# 🎳 LANE LAB

AI 볼링 스윙 분석 웹앱 — 카메라 한 대로 자세, 공 궤적, 일관성까지.

브라우저에서 바로 실행되는 PWA(설치형 웹앱)입니다. 서버 없이 모든 분석이 기기 내에서 이루어집니다.

## ✨ 주요 기능

- **실시간 자세 분석** — MediaPipe PoseLandmarker로 무릎 각도, 슬라이드 밸런스, 릴리스 타이밍, 어깨 기울기, 스텝 리듬 측정
- **공 궤적 추적** — 3구간 색상 표시(릴리스 초록 / BP 청록 / 진입 주황), 브레이크 포인트·진입각 자동 감지
- **일관성 지표** — 최근 10샷의 스피드·릴리스각·밸런스·진입각·BP 변동성을 점수화
- **RPM 추정** — 궤적 곡률 기반 회전수 추정치
- **내 라인 비교** — 평소 궤적(중앙값)과 이번 샷을 결과 화면에 오버레이
- **샷 레이더 차트** — 구속/회전/밸런스/진입각/일관성 5축 시각화
- **밝은 마커 자동 보정** — 파울라인·핀 앞에 밝은 동그라미를 두면 레인 영역 자동 인식
- **히스토리** — 샷 기록 누적, 평균/최고/일관성 요약

## 🚀 배포

정적 파일만으로 동작하므로 어떤 호스팅이든 가능합니다.

### Netlify Drop (가장 간단)
1. 이 폴더 전체를 [Netlify Drop](https://app.netlify.com/drop)에 끌어다 놓기
2. 생성된 주소를 휴대폰에서 열기

### Vercel / GitHub Pages
빌드 명령 없음, Output Directory = `.`

> 카메라는 **HTTPS**에서만 동작합니다.

## 📱 휴대폰 설치 (PWA)
- **Android Chrome**: 메뉴 → 홈 화면에 추가
- **iPhone Safari**: 공유 → 홈 화면에 추가

## 🛠 기술 스택
- **자세 인식**: Google MediaPipe Tasks Vision (Pose Landmarker, VIDEO 모드)
- **공 추적**: Canvas 픽셀 분석 (색상/움직임 기반, GPU 가속 없이 순수 JS)
- **프론트엔드**: 바닐라 JS (프레임워크 없음), Canvas 2D, Service Worker (오프라인 PWA)

## 📋 촬영 가이드
1. 후면 카메라 또는 고정 액션캠 — 선수 머리~발끝과 레인·핀이 모두 보이게
2. (선택) 파울라인과 핀 앞 바닥에 밝은 동그라미 마커 1개씩 → 자동 보정 정확도 향상
3. 공과 바닥의 색 차이가 클수록 궤적 추적이 정확
